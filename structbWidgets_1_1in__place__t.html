Detailed Description

Empty helper class to tag bwOptional constructors to use in-place construction.

In-place construction means the arguments passed to the constructor together with in_place, will be forwarded (as in, perfect forwarding) to the constructor of the type T of a bwOptional<T>.

More concrete, initializing a bwOptional like this:

bwOptional<SomeType> opt(in_place, arg1, std::move(arg2), 10.0f);

will construct an instance of SomeType as if using:

SomeType some_type_var(arg1, std::move(arg2), 10.0f);
